# WhatsApp Clone

A responsive WhatsApp Web clone built using **React.js**, **Vite**, **Firebase**, **React Router**, and **Tailwind CSS**. The application replicates the core features of WhatsApp Web, including user authentication, real-time messaging interface, chat management, and responsive UI.

## Features

- User Authentication using Firebase
- Real-time Chat Interface
- Responsive UI inspired by WhatsApp Web
- React Router based Navigation
- Firebase Firestore Database
- Firebase Storage Support
- User Profile Management
- Dark Theme Support
- Component-based Architecture
- Fast Development with Vite

## Installation

Clone the repository

```bash
git clone https://github.com/AnikDas1999/WA_Clone_react.git
```

Go to the project directory

```bash
cd WA_Clone_react
```

If your repository contains the project inside another folder:

```bash
cd wa_clone_final-main
```

Install dependencies

```bash
npm install
```

Start the development server

```bash
npm run dev
```

Open your browser

```
http://localhost:5173
```
